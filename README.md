## ðŸ“– About LangChain
[LangChain](https://www.langchain.com/) is an open-source framework that makes it easier to develop applications using LLMs.  
It provides:
- **Prompt management**
- **Chains and agents**
- **Memory modules**
- **Tool integrations** (APIs, databases, etc.)
- **Support for multiple LLM providers**
